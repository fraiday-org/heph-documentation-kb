# Heph Documentation KB

Central GitHub Pages documentation gateway for RM Native Digital enablement.

## Branching

- Default branch: `main`
- Production branch: `main`
- Commit format: `feat|fix|patch : message`

## Secret Safety

Use `.env` locally and commit only `.env.example`. Do not commit credentials, local agent files, or MCP configuration.

## Verification

Run the commands in `docs/SETUP.md` and keep the tree clean before pushing.
